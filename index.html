5imple

5imple is a javascript API that eases the development of HTML5's canvas element. It allows you to be more productive and efficient when writing your code, getting more done quicker. Take this code for example, the following is what you would write to draw three circles onto your canvas.

var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");

ctx.beginPath();
ctx.arc(100,100,100,0,2*Math.PI);
ctx.fillStyle = "#00CCFF";
ctx.fill();
ctx.beginPath();
ctx.arc(100,100,50,0,2*Math.PI);
ctx.fillStyle = "#0011FF";
ctx.fill();
ctx.beginPath();
ctx.arc(100,100,25,0,2*Math.PI);
ctx.fillStyle = "#FFCC00";
ctx.fill();

This can't be an efficient way to write code for the canvas, so lets change it up. Here's the code using 5imple.

useCanvas("myCanvas");
myCircle = new Circle(100,100,100,"#00CCFF","myCircle");
myCircle2 = new Circle(100,100,50,"#0011FF","myCircle2");
myCircle3 = new Circle(100,100,25,"#FFCC00","myCircle3");
updateCanvas();

Hey look! You can actually read it! Plus, your circles and such are now objects, so you can easily change their position without having dozens of x and y variables in your script. updateCanvas() renders the whole canvas for you, so you don't have to redraw the arc and set the color and fill it yourself. You tell it to update and it goes through all of your objects and renders them!
